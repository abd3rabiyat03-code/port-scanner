How to Run It
1- You must have VS Code installed.
2- You must have Python installed on your computer. You can verify this by going to cmd and typing `py`.
3- After verifying and completing these two steps, you must open the file in VS Code.
4- Click Run to run the code.
5- The program will then ask you to enter the target IP address and then the range, for example (1-100).
Important Warning: Do not run the code to scan any public port.
This code scans open ports for any IP address or domain you specify and then saves the results to an external file. This code uses the ThreadPoolExecutor feature to make port scanning faster.
